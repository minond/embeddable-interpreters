Embeddable interpreters. The idea is that you could take one of these interpreters and embed them into web apps. You'll mostly find esoteric programming languages.

Resources for Brainfuck:
  - https://esolangs.org/wiki/Brainfuck
  - https://en.wikipedia.org/wiki/Brainfuck

Building and testing:

  make
  ./main brainfuck test/bf/helloworld.bf
